# PDF工具箱

一个功能强大的PDF处理工具集，提供各种PDF文件的处理功能，包括合并、分割、压缩、转换、编辑等。

## 功能列表

### 基础操作
- **PDF合并**：将多个PDF文件合并为一个PDF文件
- **PDF分割**：将PDF文件分割为多个部分
- **PDF压缩**：减小PDF文件大小，优化存储和传输
- **PDF加密/解密**：为PDF文件添加密码保护或移除密码保护

### 转换功能
- **PDF转Word**：将PDF文件转换为可编辑的Word文档
- **PDF与图片互转**：PDF转图片或将多张图片合并为PDF
- **PDF转文本**：从PDF文件中提取文本内容
- **Word转PDF**：将Word文档转换为PDF格式
- **Excel转PDF**：将Excel表格转换为PDF格式
- **PPT转PDF**：将PowerPoint演示文稿转换为PDF格式

### 编辑功能
- **PDF旋转**：旋转PDF页面方向
- **元数据编辑**：编辑PDF文件的元数据信息
- **PDF水印**：为PDF文件添加文字或图片水印
- **PDF页眉页脚**：为PDF文件添加页眉和页脚
- **PDF书签**：为PDF文件添加或编辑书签
- **PDF注释**：为PDF文件添加注释、高亮和下划线
- **PDF表单**：填写或创建PDF表单

### 高级功能
- **PDF比较**：比较两个PDF文件的差异
- **OCR识别**：识别扫描PDF中的文字内容
- **批量处理**：批量处理多个PDF文件，支持多种操作组合
- **PDF优化**：优化PDF文件结构和性能，减小文件大小
- **PDF签名**：为PDF文件添加数字签名

## 安装方法

### 系统要求
- Python 3.6 或更高版本
- Windows、macOS 或 Linux 操作系统

### 安装步骤
1. 克隆或下载本项目到本地
2. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```bash
   python main.py
   ```

## 使用说明

1. **添加文件**：在左侧"文件操作"区域，点击"添加PDF文件"按钮选择要处理的PDF文件
2. **选择功能**：从左侧功能菜单中选择所需的功能模块
3. **设置参数**：根据功能要求设置相关参数
4. **执行操作**：点击"执行操作"按钮完成处理
5. **查看结果**：处理完成后，查看输出结果或保存到指定目录

## 依赖库

- PyPDF2：用于PDF文件的读取和写入
- tkinter：用于创建图形用户界面
- PIL (Pillow)：用于图像处理
- python-docx：用于Word文档处理（可选）
- openpyxl：用于Excel文档处理（可选）
